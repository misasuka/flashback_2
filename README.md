# Flashback Timeline（离线单页小游戏）

一个受 NYT Flashback 启发的离线网页小游戏：
**把历史事件插入到正确的时间轴位置**。

## 功能说明

- 保留核心玩法：按时间排序插入事件
- 单页应用，纯前端（HTML/CSS/JavaScript）
- 使用本地题库：`data/events.json`（50 条西方历史事件，文艺复兴到二战结束）
- 每轮随机抽取 9 题：
  - 先给 1 个基准事件
  - 再依次出现其余 8 个事件供玩家插入
- 通过点击时间轴中的“插入到这里”进行作答
- 答对 +1 分，答错 +0 分
- 实时显示当前分数与已完成题数
- 结束后展示结果页，支持“重新开始”

## 项目结构

```text
.
├── index.html             # 页面结构
├── styles/
│   └── main.css           # 样式（简洁现代风格）
├── src/
│   ├── app.js             # 入口：初始化游戏与 UI
│   ├── game.js            # 核心玩法逻辑（抽题、判定、计分）
│   ├── ui.js              # 视图渲染与交互绑定
│   └── utils.js           # 工具函数（洗牌、日期格式化等）
└── data/
    ├── events.json        # 本地 JSON 题库（50 题）
    └── events.js          # 供浏览器模块直接导入的数据镜像
```

## 本地运行

### 方式 1：直接双击打开（离线）

直接用浏览器打开 `index.html` 即可开始游戏。

### 方式 2：本地静态服务（可选）

如果你更习惯本地服务，可在项目目录运行：

```bash
python -m http.server 8000
```

然后访问：<http://localhost:8000>

## 规则补充

- 时间轴会始终按真实时间顺序展示（答错后也会自动归位到正确位置）
- 每轮满分为 8 分（因为首个基准事件不计分）
